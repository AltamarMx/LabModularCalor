# Dispositivo de adquisición y visualización de datos

Es una RaspBerry Pi con una plataforma IoT instalada y configurada como servidor. Es útil en caso de no contar con internet y una conexión inalámbrica, se puede utilizar este dispositivo para que sirva de adquisición y visualización de datos en tiempo real. Si se cuenta con internet, se puede solicitar la alta del dispositivo al correo gbv@ier.unam.mx para que se proporcione una liga a un tablero de visualización de datos.
